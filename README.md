# PlayRPG

## Install requirements
```bash
pip install -r requirements.txt
```

## Set up API keys
Set environment variables:
- `GEMINI_API_KEY`
- `ELEVENLABS_API_KEY` (required only if you want BGM generation)

## Run server
```bash
python server.py
```
